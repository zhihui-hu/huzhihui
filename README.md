# 胡志辉 / Hu Zhihui

写代码，也写一点东西。

从 2016 年 12 月开始做软件开发，先后参与过金融和医疗行业的系统建设与产品交付。现在主要在看 AI 应用、Web 开发、移动端和跨平台这几件事。

我对“把功能做出来”当然有兴趣，但更在意另一半：流程顺不顺，数据准不准，权限清不清，系统能不能稳稳落地。
